# CIFAKE Real vs AI-Generated Image Classification

Proyek ini melakukan klasifikasi **gambar REAL vs gambar AI-generated (FAKE)** menggunakan **ResNet101** dan dataset **CIFAKE**. Dataset ini berisi gambar asli dari CIFAR-10 dan citra sintetis yang dibuat oleh model generatif AI, jadi tugasnya adalah binary classification antara *real* dan *fake*. :contentReference[oaicite:1]{index=1}
